Tricks for adding Background Music in your Webpage:


1. Using the HTML Audio Element

BGsound attribute is no longer used by the browsers

2. Using the HTML Audio Element

In these audios can be added from the local storage of computers/ harddrives

3. Using the Embed Element

If you want to use youtube video as an audio on your webpage.

1. Firstly open the video, click on share and select the embed option, from there you can copy the code of using the youtube video in your webpage.

2. Secondly, to just hear the music, set width and height to =1, due to which video would no longer be visible but music would continue playing in the background, online!
